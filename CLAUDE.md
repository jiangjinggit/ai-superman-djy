# AI超人成长计划 - 项目说明

## 项目定位

这是一个长期的个人成长+自媒体运营项目，目标是记录从AI小白成长为**AI推广专家+AI教学专家+自媒体专家**的完整过程。

## 核心理念

**螺旋式成长模型**：学习 -> 实践 -> 总结+分享 -> 继续学习 -> 实践 -> 总结+分享

## 内容方向

### AI推广+AI教学（由浅入深）

1. **AI科普**：让普通人理解AI是什么、能做什么
2. **小白入门**：小白用户如何学习和使用AI
3. **专业应用**：
   - 软件开发（AI辅助编程）
   - 设计领域（AI设计工具）
   - 自媒体领域（AI内容创作）
   - 办公效率（AI办公助手）
4. **超级个体**：如何用AI成为一个人的军队

### 内容形式

- 短篇图文（适合快速阅读）
- 短视频（适合碎片化传播）
- 长视频教程（适合深度学习）

### 发布平台

**国内平台**（第一阶段）：
- 微信公众号
- 小红书
- 掘金
- B站
- 抖音

**海外平台**（后续扩展）：
- YouTube
- Twitter/X
- Medium

## 更新频率

- 启动阶段：日更
- 后续根据数据反馈调整

## 内容风格规范

### 文章风格：轻松幽默

所有内容创作必须遵循以下风格要求：

1. **语气轻松**：像朋友聊天一样，不要端着、不要说教
2. **适度幽默**：加入段子、调侃、自嘲，让内容有趣
3. **接地气**：用大白话，避免过于专业的术语堆砌
4. **有个人特色**：展现真实的自己，包括踩坑、吐槽、小确幸
5. **节奏明快**：短句为主，不拖沓

**风格示例**：
- ❌ "人工智能技术的快速发展为我们带来了前所未有的机遇"
- ✅ "AI这玩意儿，说实话，一开始我也觉得离自己很远，直到它帮我5分钟写完了周报..."

- ❌ "本文将系统性地介绍AI工具的使用方法"
- ✅ "今天教你几个AI骚操作，学会了老板都得叫你大佬"

## 图片生成规范

### 所有配图必须提供AI生图提示词

每篇内容的配图部分需要包含完整的AI生图提示词，格式如下：

```
## 配图提示词

### 图1：[图片用途描述]
**中文描述**：[图片内容的中文描述]
**英文提示词**：[用于Midjourney/DALL-E的英文提示词]
**风格参数**：[可选，如 --ar 3:4 --style raw]
```

### 提示词规范

1. **统一视觉风格**：整个账号保持一致的视觉调性
2. **默认风格**：扁平插画风格、科技感、蓝紫色调
3. **必须包含**：
   - 图片主体描述
   - 风格关键词（flat illustration, minimalist, tech style等）
   - 色调关键词（blue and purple gradient, vibrant colors等）
   - 尺寸比例（根据平台调整）

### 各平台图片尺寸

| 平台 | 封面尺寸 | 内容图尺寸 |
|-----|---------|-----------|
| 小红书 | 3:4 或 1:1 | 3:4 |
| 公众号 | 2.35:1（头图） | 不限 |
| 抖音 | 9:16 | 9:16 |
| B站 | 16:9 | 16:9 |
| 掘金 | 2:1 | 不限 |

### 自动图片生成工具

项目集成了Gemini API自动生成配图的功能。

**配置步骤**：
1. 复制 `.env.example` 为 `.env`
2. 在 `.env` 中填入你的 Gemini API Key
3. 运行 `npm install` 安装依赖

**使用方法**：
```bash
# 生成指定文章的配图
npm run gen content/articles/2026-01-01-wechat-用AI重塑自己.md

# 批量生成所有文章的配图
npm run gen:all
```

**工作原理**：
1. 脚本读取文章中 `## 配图提示词` 部分
2. 提取每个图片的英文提示词
3. 调用Gemini API生成图片
4. 保存到 `content/assets/YYYY-MM-DD/` 目录

**注意**：
- 每张图片约 $0.039
- 生成间隔2秒，避免API限流
- 图片保存为PNG格式
