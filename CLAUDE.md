# AI超人成长计划 - 项目说明

## 项目定位

这是一个长期的个人成长+自媒体运营项目，目标是记录从AI小白成长为**AI推广专家+AI教学专家+自媒体专家**的完整过程。

## 核心理念

**螺旋式成长模型**：学习 -> 实践 -> 总结+分享 -> 继续学习 -> 实践 -> 总结+分享

## 内容方向

### AI推广+AI教学（由浅入深）

1. **AI科普**：让普通人理解AI是什么、能做什么
2. **小白入门**：小白用户如何学习和使用AI
3. **专业应用**：
   - 软件开发（AI辅助编程）
   - 设计领域（AI设计工具）
   - 自媒体领域（AI内容创作）
   - 办公效率（AI办公助手）
4. **超级个体**：如何用AI成为一个人的军队

### 内容形式

- 短篇图文（适合快速阅读）
- 短视频（适合碎片化传播）
- 长视频教程（适合深度学习）

### 发布平台

**国内平台**（第一阶段）：
- 微信公众号
- 小红书
- 掘金
- B站
- 抖音

**海外平台**（后续扩展）：
- YouTube
- Twitter/X
- Medium

## 更新频率

- 启动阶段：日更
- 后续根据数据反馈调整

## 开发规范

### 文件命名规范

- 文章：`YYYY-MM-DD-平台-标题简述.md`
- 视频脚本：`YYYY-MM-DD-video-标题简述.md`
- 周计划：`YYYY-WXX-weekly-plan.md`

### 内容创作流程

1. 选题（参考选题库和热点）
2. 大纲（先列框架）
3. 初稿（AI辅助+人工润色）
4. 配图/录制
5. 发布
6. 数据追踪

### 标签体系

内容标签用于分类和检索：
- `#AI科普` `#AI入门` `#AI编程` `#AI设计` `#AI办公`
- `#工具推荐` `#实战教程` `#经验分享` `#行业观察`
- `#小红书` `#公众号` `#掘金` `#B站` `#抖音`
