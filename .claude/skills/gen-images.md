# /gen-images - 生成文章配图

根据文章中的 `## 配图提示词` 部分，使用 MCP Playwright 打开 Gemini 网页生成配图。

## 触发格式

```
/gen-images
/gen-images content/articles/2026-01-09-wechat-xxx.md
/gen-images --all
```

## 使用方式

1. **生成当前文章配图**：直接输入 `/gen-images`，自动识别 IDE 中打开的文章
2. **生成指定文章配图**：`/gen-images <文章路径>`
3. **批量生成所有文章**：`/gen-images --all`

## 执行步骤

### 步骤1：读取文章配图提示词

从文章的 `## 配图提示词` 部分提取每张图片的：
- 图片名称/用途
- 英文提示词
- 尺寸比例（--ar 参数）

### 步骤2：用 MCP Playwright 生成配图

1. 打开 Gemini (https://gemini.google.com)
2. 逐张输入提示词生成图片
3. 下载图片到 `.playwright-mcp/` 临时目录
4. 整理到 `content/assets/日期/` 目录
5. 按序号重命名（图1.png、图2.png...）

### 步骤3：返回完成报告

```
✅ 配图生成完成

文章：content/articles/2026-01-09-wechat-xxx.md
图片：content/assets/2026-01-09/ (4张)

请检查图片质量。
```

## 关键质量要求

### ⚠️ 图片与文章内容必须匹配

- 如果文章讲的是**国产AI工具**（豆包、通义、Kimi等），图片中也必须体现国产工具
- 提示词要明确指定文章中提到的具体工具名称
- 避免生成通用的"AI"图片，要有针对性

### 配图提示词规范

1. **具体化**：明确提及文章中的具体工具/概念名称
2. **中文元素**：需要时包含中文文字
3. **风格统一**：扁平插画风格，蓝紫色渐变主题
4. **比例正确**：
   - 小红书：竖版 3:4
   - 公众号/掘金：横版 16:9
