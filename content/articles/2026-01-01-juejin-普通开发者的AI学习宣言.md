## 第一天的小实验

既然是Day 1，总得做点什么吧。

我试着用Claude帮我生成了一个简单的工具函数：

**我的提示词：**
> 帮我写一个JavaScript函数，用于格式化文件大小，输入字节数，输出带单位的字符串，比如 1024 -> "1 KB"

**Claude的回答：**

```javascript
function formatFileSize(bytes) {
  if (bytes === 0) return '0 Bytes';

  const k = 1024;
  const sizes = ['Bytes', 'KB', 'MB', 'GB', 'TB'];
  const i = Math.floor(Math.log(bytes) / Math.log(k));

  return parseFloat((bytes / Math.pow(k, i)).toFixed(2)) + ' ' + sizes[i];
}

// 测试
console.log(formatFileSize(1024));      // "1 KB"
console.log(formatFileSize(1234567));   // "1.18 MB"
console.log(formatFileSize(1073741824)); // "1 GB"
```

说实话，这个函数我自己写也能写出来，但可能要5分钟。Claude秒回。

而且它考虑到了边界情况（0字节），我可能第一版都想不到。

## 一点小感悟

Day 1，只是一个开始。

但有几点感受想分享：

1. **AI确实能提效，但不是万能的**。它能帮你写代码，但你得知道要写什么代码。
2. **提示词很重要**。问得越清楚，答案越靠谱。
3. **最难的是坚持**。用一次两次不难，难的是形成习惯。

接下来的364天，我会继续记录这个过程。

希望365天后回来看这篇文章，是成长而不是打脸。
