# 视频脚本模板

## 短视频脚本（15-60秒）- 抖音

### 结构：开头hook + 核心内容 + 结尾引导

```
【0-3秒】开头Hook
- 直接抛出痛点/惊人事实/问题
- 例："你还在手动写代码？用这个AI工具，效率翻10倍"

【4-45秒】核心内容
- 快速展示方法/技巧/工具
- 用实操画面+旁白
- 重点用文字强调

【46-60秒】结尾
- 总结价值
- 引导互动："试试看，有问题评论区见"
- 引导关注："关注我，每天分享AI技巧"
```

### 抖音脚本示例

```
【画面】AI生成代码的屏幕录制

【旁白】
还在一行行敲代码？太慢了！
```

## 中长视频脚本（5-15分钟）- B站

### 结构

```
【0:00-0:30】开场
- 自我介绍
- 本期主题
- 内容预告

【0:30-2:00】背景/问题
- 为什么要讲这个
- 观众痛点

【2:00-10:00】主体内容
- 分点讲解
- 实操演示
- 案例展示

【10:00-12:00】进阶技巧/注意事项
- 高级用法
- 常见问题

【12:00-13:00】总结
- 核心要点回顾
- 适用场景

【13:00-15:00】结尾
- 互动引导
- 下期预告
- 关注提醒
```

## 脚本写作技巧

### 开头黄金3秒
- 直接抛出价值/痛点
- 不要废话铺垫
- 用问题/数据/对比抓眼球

### 内容节奏
- 每30秒一个信息点
- 避免长段落念白
- 用画面转场保持节奏

### 口播技巧
- 口语化，不要书面语
- 短句为主，便于断句
- 重点信息重复强调

### 引导互动
- 开头：先点关注
- 中间：屏幕上发弹幕
- 结尾：一键三连

---

## 封面设计要点

- 文字要大，手机能看清
- 3-5个字概括核心内容
- 使用对比色增加吸引力
- 人物出镜可增加信任感
