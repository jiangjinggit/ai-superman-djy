## 使用说明

1. 复制此模板，按格式填写内容
2. 运行 `npm run video <脚本路径>` 自动生成视频
3. 生成的视频保存在 `content/assets/日期/video/` 目录

### 命令示例

```bash
# 生成双平台视频
npm run video content/scripts/2026-01-01-video-xxx.md

# 只生成抖音版
npm run video content/scripts/2026-01-01-video-xxx.md --douyin

# 只生成B站版
npm run video content/scripts/2026-01-01-video-xxx.md --bilibili
```

### 注意事项

- 每个场景的配音文案会合并生成一段完整的音频
- 图片会按场景顺序展示，时长根据音频自动分配
- B站版本会自动调整为16:9横屏（图片居中+模糊背景）
